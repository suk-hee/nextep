KLDS 디자인 시스템의 Header 컴포넌트입니다.
Figma node 1:17439 (company=posco variant)를 기반으로 구현되었으며, button-icon, avatar, mode-switch 컴포넌트를 재사용합니다.

기본 헤더 (company=posco)

주요 구성 요소

  • Left Section button-icon 컴포넌트 (48×48, header__btn-menu 래퍼) + 즐겨찾기 사용자 그룹 (avatar 컴포넌트)
  • Center Logo Next EP 로고 (중앙 정렬, absolute 위치)
  • Mode Switch mode-switch 컴포넌트 — 클래식/AI 모드 전환
  • Right Buttons button-icon 컴포넌트 × 4 (검색, 빠른실행, 알림, 설정) + 사용자 프로필
  • 아이콘 색상 mask-image 방식 — background-color로 정확한 색상 제어, hover 시 #F3F4F6 배경

디자인 토큰 매핑

  • 색상 #FFFFFF → --klds-color-white, #005386 → --klds-color-button-fill-surface, #D6E4FF → --klds-color-blue-blue02, #F3F4F6 → --klds-color-background-hover
  • 간격 4px → --klds-gap-xxs, 8px → --klds-gap-xs, 12px → --klds-padding-s, 16px → --klds-gap-m, 20px → --klds-padding-l
  • 높이 20px → --klds-size-size3, 24px → --klds-size-size4, 40px → --klds-size-size6, 48px → --klds-size-size7, 56px → --klds-size-size8
  • 반경 20px → --klds-radius-l, 9999px → --klds-radius-circle
  • 타이포그래피 Pretendard 16px/600 → --klds-typography-fontsize-m / font-weight: 600

button-icon 적용 개선점

  • 컴포넌트 재사용 button-icon.css 공통 컴포넌트 활용 → 코드 중복 제거
  • 정확한 크기 메뉴 버튼: 48×48px (--klds-size-size7), 기능 버튼: 40×40px (--klds-size-size6), 아이콘: 24×24px (--klds-size-size4)
  • mask-image 방식 아이콘 색상을 background-color로 제어 → 피그마와 정확히 일치
  • hover 효과 0.15s transition, 배경색 #F3F4F6 (--klds-color-background-hover)
  • 일관성 모든 아이콘 버튼이 동일한 스타일 시스템 사용
